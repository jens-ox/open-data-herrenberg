# Open Data Portal Herrenberg

Herzlich willkommen auf dem Open Data Portal der Stadt Herrenberg. Hier finden Sie eine Vielzahl von Daten, die von der Stadt Herrenberg und ihren Partnern bereitgestellt werden. Die Daten sind frei verfügbar und können von jedermann genutzt werden. Wir freuen uns, wenn Sie die Daten für Ihre eigenen Projekte verwenden und freuen uns über Ihr Feedback.
